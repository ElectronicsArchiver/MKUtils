### How to use

1. Add the JitPack repository to your build.gradle.kts

    ```kotlin
    repositories {
        maven { url = uri("https://www.jitpack.io") }
    }
    ``` 
2. Add the [dependency][Jitpack]

    ```kotlin
     dependencies {
        implementation("com.github.plumygame:mkutils:<module>:<version>")
    }
    ```
